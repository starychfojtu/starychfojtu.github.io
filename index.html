Problém

  • chceme funkci co k nullable vstupu přičte 5 a vydělí druhým parametrem
  • pokud se něco pokazí, vrátí odpověď na všechno

public T F(int? input, int divider)
{
    var x = input is null ? null : input.Value + 5;
    var y = x is null || divider == 0 ? null : x.Value / divider;
    return y is null ? 42 : y.Value;
}
  • v error handlingu se ztrácí myšlenka

Jiná možnost

public T F(int? input, int divider)
{
    var default = 42;
    if (input is null || divider == 0)
    {
        return default;
    }

    return (input.Value + 5) / divider;
}
  • musíme na začátky vědět jak se všechno může pokazit (zapomeneme a duplikujeme)

    from valueA in a

Validation


public ITry<Person, Error[]> Create(string name, string email) =>
    Try.Aggregate(
        ValidateName(name),
        ValidateEmail(email)
        (n, e) => new Person(n, e)
    );

private ITry<Name, Error[]> ValidateName(string name);
private ITry<Email, Error[]> ValidateEmail(string email);
private ITry<Phone, Error[]> ValidatePhone(string phone);

private ITry<Name, Error[]> ValidateName(string name) =>
    name.IsNullOrEmpty
        ? Try.Error<Name, Error[]>(Error.NameCannotBeEmpty)
        : Try.Success<Name, Error[]>(new Name(name));

More functional example

public IOption<Person> GetById(Guid id);
public ITry<Unit> DoSomething(Person person);

var person = PersonRepository.GeById(id);
var result = person.Map(p => PersonService.DoSomething(p));
return result.Match(
    some => some.Match(
            success => HttpOk(),
            error => HttpBadRequest()
        ),
    _ => HttpNotFound()
);
